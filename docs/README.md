# 메뉴 추천

## 요구 사항

- 메뉴
  - 한 주에 같은 메뉴는 두 번 이상 먹지 않는다.
  - 같은 카테고리는 최대 2회까지만 고를 수 있다
  -> 즉, 모든 코치는 매일 다른 음식을 먹어야 하며, 같은 카테고리는 최대 2번까지 허용된다.

## 어플리케이션 흐름

1. 코치의 이름을 입력받는다.
   - "코치의 이름을 입력해 주세요. (, 로 구분)"
       - 코치는 최소 2명, 최대 5명
       - 코치의 이름은 최소 2글자, 최대 4글자
       - 코치의 이름은 중복되지 않는다.
2. 각 코치가 못 먹는 메뉴를 입력받는다.
   - "~~ 이(가) 못 먹는 메뉴를 입력해 주세요." -> , 로 구분
       - 코치는 최소 0개, 최대 2개의 못 먹는 메뉴가 있다.
       - 먹지 못하는 메뉴가 없으면 빈 값을 입력한다.
3. 메뉴를 추천한다.
   - 구분 : 요일 출력
   - 카테고리 : 한식, 일식, 중식, 양식, 아시안 중 선택된 카테고리 출력
   - 코치 이름 : 코치가 선택한 메뉴 출력
4. 프로그램을 종료한다.
   - 추천을 완료했습니다.
